## 🔗 프로젝트 구현 내용

<br>

**member CRUD**
- Spring MVC 아키텍처를 활용하여 Upcycling CRUD(Create, Read, Update, Delete) 기능을 구현
- Controller에서 HTTP 요청을 처리하여 모델과 뷰를 연결하였으며, 각각의 요청에 대응하는 핸들러 메서드를 정의하여 비즈니스 로직을 수행
